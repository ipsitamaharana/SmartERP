# SmartERP

A cloud-based ERP system inspired by Tally ERP 9.

## Tech Stack

### Frontend
- Next.js
- React.js
- Tailwind CSS
- ShadCN UI
- TanStack Table
- React Hook Form

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL / Supabase

## Features

- Company Management
- Ledger Management
- Stock Management
- Purchase Voucher
- Sales Voucher
- Billing System
- Reports
- Keyboard Navigation
